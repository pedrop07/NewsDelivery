# News Delivery

Um site de listagem de notícias, onde é possível criar, editar e excluir notícias!

## Stack utilizada

 - [React](https://react.dev)
 - [React Router Dom](https://reactrouter.com/en/main)
 - [Vite](https://vitejs.dev)
 - [TypeScript](https://www.typescriptlang.org)
 - [TailwindCSS](https://tailwindcss.com)
 - [React Query](https://tanstack.com/query/v3/)
 - [React Hook Form](https://react-hook-form.com)
 - [Zod](https://zod.dev)
 - [Axios](https://axios-http.com/ptbr/docs/intro)
 - [Phosphor React](https://phosphoricons.com)

## Rodando localmente

Para rodar a aplicação localmente em sua máquina, você devera seguir os seguintes passos:

1. Primeiramente, para que o front-end da aplicação funcione corretamente, é necessário que o back-end esteja sendo executado, pois será por meio dele que o front-end consumirá a API. Se você ainda não está com o back-end em execução, acesse a pasta `server` deste repositório e siga o tutorial.

2. Agora que o back-end está em execução, é necessário ter o [Node.js](https://nodejs.org/en) instalado em sua máquina, juntamente com um gerenciador de pacotes, como NPM ou Yarn.

3. Abra o seu terminal de preferência e execute a seguinte sequência de comandos:

```bash
  git clone https://github.com/pedrop07/NewsDelivery.git
```

Entre no diretório do projeto.

```bash
  cd NewsDelivery/client
```

Instale as dependências.

```bash
  npm install
```

Gere uma versão de build da aplicação.

```bash
  npm run build
```

Execute a versão de build criada.

```bash
  npm run preview
```
