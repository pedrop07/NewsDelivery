
# NewsDelivery

Uma aplicação full stack de um CRUD de notícias.

### Instalação

- Acesse a pasta `server` primeiramente para obter acesso ao passo a passo de como executar a API.

- Após ter a API em execução e funcionando corretamente, acesse a pasta `client` para obter acesso ao passo a passo de como executar o lado do cliente da aplicação.
